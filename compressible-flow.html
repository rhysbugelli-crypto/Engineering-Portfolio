About the Tool

Built to support compressible aerodynamics coursework, this calculator implements the core relations from gas dynamics — from subsonic isentropic flow through to supersonic shock analysis and expansion fans.

All calculations run client-side in the browser with no dependencies beyond Plotly for the flow visualisations. Input any Mach number or flow condition and the tool returns the full set of associated parameters instantly.
